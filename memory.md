# Memory

Persistent context for agents and humans working in this repo.

## Project

- **tiep-os** - a personal operating system of automated agents and skills
- Built to scale: each skill lives in its own directory under `skills/`
- Currently focused on Polymarket trading signals

## Decisions

- Skills are organized by domain (e.g., `skills/polymarket/`, `skills/finance/`)
- Each skill is self-contained with its own dependencies and entry point
- Shared utilities go in `lib/` when needed
